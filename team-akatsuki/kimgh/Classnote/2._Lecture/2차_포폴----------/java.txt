====== java ======
Created 화요일 21 7월 2020

C++ 기반으로 개발, 가전제품 가동하는 목적으로 쓰였지만
다양한 운영체제 아래에서 이식성이 높아서 www 개발 후 급성장

JVM(Java Virtual Machine)을 통해서 실행함. 그 내부의 샌드박스에서 구동됨.
운영체제가 직접 실행하는 것이 아니라 속도는 비교적 느릴 수도 있으나, 보안은 강력함.
중립성도 높다. 어디서든 똑같이 작동한다. 특정 플랫폼을 표적하여 컴파일되는 것이 아니라
바이트 코드라는 동일한 포맷으로 변화되기 때문에 JVM만 지원되면 동일하게 실행 가능
