Github 다운로드



=====================
Github

마이크로소프트가 인수를 했다. 파일이 올라가는 것처럼 보이지만 변경 내역이 올라감. 
GIT을 사용하려면, 자신이 누구인지 인증이 필요함. PC에 Git을 설치하고 이름과 이메일로 개별 구분을 함. 
버전을 생성해서 업로드하고 나서 집에서 Git을 설치하고 본인의 이름을 바꿔서 올리면, 
학원에서 수정한 내용과 집에서 수정한 내용 사이의 차이를 볼 수 있다!

Github는 Git으로 만드는 버전을 관리해주는 서비스지 Git 자체는 아님.

Git은 변경 내역에 version을 지정해주는 프로그램. Github는 Git으로 지정된 version을 업로드하고 나눌 수 있는 허브.
