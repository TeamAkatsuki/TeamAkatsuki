====== git ======
Created 화요일 21 7월 2020

🏁️버젼 관리 프로그램.
 
--------------------

working directory = repository 작업 대상인 파일들
index (staging area로 준비영역) = 관리 대상 목록
head = commit 완료된 확정본

--------------------

파일 변경 
-> add로 커밋할 파일을 지정함 
-> 변경 내역은 staging area로 
-> commit 
-> 서버로 push

--------------------

git init을 해서 초기 작업을 생성하고

1차 커밋

2차 커밋

2차 커밋 이후 파일을 애드함

마지막 커밋 상태로 돌리는 것을 reset이라 함.
